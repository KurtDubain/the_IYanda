作品介绍

本网页主要是基于HTML&CSS&JavaScript技术设计完成的

我们还引入jQuery库来便捷我们对页面里的交互设计进行进一步完善

页面主要基于Flex布局以使背景规格统一，使整体布局风格更加相近

本作品由五个页面组成，分别为首页、全部课程展示页面、重要课程展示页面、团队介绍页面以及本说明页面

我们设计的宗旨是“简洁不简单”

通过添加hover动效、CSS3动画以及jQuery事件触发，来让整体观感更加舒适，并且展示重要信息

具体说明信息请看本作品中readme文件，感谢理解
